Welcome to BodLanes

This presentation is designed to be viewed at exactly 1920 × 1080 pixels, which is fullscreen (F11) on a horizontally-oritented "full HD" monitor. You can simulate this in a modern web browser like Google Chrome using the Dev Tools. BodLanes presentations are HTML files enhanced with special <bl-*> tags to represent special kinds of content and are suitable for deployment to web-technology-powered digital signage and kiosks.

Use the links in the navigation below to explore this example presentation and learn more about what BodLanes can do, or make your own presentation in the presentations folder by creating a folder and index.html file there, e.g. presentations/my-presentation/index.html.

Fourth Page

This page 'identifies as' the third page, so the menu still shows that we're on the third page. This is useful for pages that are clear 'children' of others, such as in galleries with additional information about each item.

There's also a <bl-floater> on this page: a block type that can be positioned anywhere you like: in this case, it's being used to put the "back" button in the top right corner. Give it a click!
