Example Two

As well as demonstrating additional techniques such as the use of column (rather than row) menus, fontawesome icons, and the updating of multiple <bl-container>s at once, this presentation contains tips on creating and deploying presentations using BodLanes2.

Touch the 'edit' icon on the menu, to the left, to get started.

Editing presentations

Presentations are folders in the presentations folder (or in the case of this one, the examples folder), e.g. presentations/my-presentation. As a minimum, they should contain an index.html file: this is a standard HTML5 file (although you'll probably want to add a few special elements that you might not yet be familiar with).
